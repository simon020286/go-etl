Pipeline Management

GET /api/v1/pipelines - List all pipelines
Query params: ?offset=0&limit=20&state=RUNNING&enabled=true&order_by=created_at&order_dir=desc
POST /api/v1/pipelines - Create new pipeline
Body: {"name": "string", "description": "string", "config_yaml": "string", "enabled": true}
GET /api/v1/pipelines/{id} - Get pipeline by ID
PUT /api/v1/pipelines/{id} - Update pipeline
Body: {"name": "string", "description": "string", "config_yaml": "string", "enabled": true}
DELETE /api/v1/pipelines/{id} - Delete pipeline
GET /api/v1/pipelines/{id}/status - Get pipeline status

Pipeline Control

POST /api/v1/pipelines/{id}/start - Start pipeline execution
Optional body: {"trigger_type": "manual", "trigger_data": "string"}
POST /api/v1/pipelines/{id}/stop - Stop pipeline execution
POST /api/v1/pipelines/{id}/pause - Pause pipeline execution
POST /api/v1/pipelines/{id}/resume - Resume paused pipeline
GET /api/v1/pipelines/running - Get all running pipelines

Execution Management

GET /api/v1/executions - List executions
Query params: ?offset=0&limit=20&pipeline_id=1&status=COMPLETED
GET /api/v1/executions/{id} - Get execution details
GET /api/v1/executions/{id}/logs - Get execution logs
GET /api/v1/pipelines/{id}/stats - Get pipeline statistics

System

GET /api/v1/health - Health check
GET /ws - WebSocket endpoint for real-time updates
